# Horarios

### Ejecutar

```bash
python3 horarios.py
```

### Disclamer

En esta version el script usara la zona horaria de Tokyo para obtener los horarios de algunos paises de latinoamerica.

Si deseas cambiar la zona horaria modifica el archivo modules/settings.py y edita la variable 'zone_to_compare'
Si deseas agregar o quitar ciudades modifica el archivo modules/zones.py

### Formato para agregar la hora

Para convertir la hora debes poner el formato de 'hh'.
Ejemplo: 18

### Formato para agregar la hora y minutos

Para convertir la hora y minutos debes poner el formato de 'hh:mm'.
Ejemplo: 18:30

### Formato para agregar la fecha

Para agregar la fecha debes usar el formato 'mm-dd'.
Ejemplo: 02-18

* Pero si deseas no agregar la fecha puedes dejarlo en blanco.
